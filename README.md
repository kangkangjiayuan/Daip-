# 川小农聊天软件 - 视频播放功能使用说明

## 功能介绍

现在聊天软件支持使用 `@电影` 命令直接在聊天界面中播放腾讯视频！用户只需在聊天中分享腾讯视频链接，系统会自动解析并显示嵌入式播放器。同时，您还可以使用 `@电影` 命令随机播放免费电影资源。

## 使用方法

### 1. 在聊天中分享腾讯视频

在聊天输入框中输入以下格式的命令：

```
@电影 腾讯视频URL
```

例如：
```
@电影 https://v.qq.com/x/cover/mzc00200d70bilg/g4101f350j1.html
```

### 2. 随机播放免费电影

直接输入命令，无需提供URL：

```
@电影
```

系统会自动为您查找并播放一个随机的免费电影。

### 3. 系统自动解析并播放

发送后，系统会自动：
- 识别腾讯视频URL或执行随机电影查找
- 提取视频ID或获取随机电影信息
- 生成嵌入式播放器
- 在聊天界面中直接显示视频

### 4. 支持的腾讯视频URL格式

系统支持多种腾讯视频URL格式：
- `https://v.qq.com/x/cover/xxx/yyy.html`
- `https://v.qq.com/x/page/xxx.html`
- `https://v.qq.com/detail/m/xxx.html`

## 功能特点

✅ **自动解析** - 无需手动转换URL，系统自动处理

✅ **直接播放** - 在聊天界面中嵌入式播放，无需跳转

✅ **响应式设计** - 在桌面端显示高清画质，移动端自适应

✅ **多种格式支持** - 支持常见的腾讯视频URL格式

✅ **优雅提示** - 对于不支持的视频链接，系统会给出友好提示

✅ **免费电影推荐** - 自动获取并推荐腾讯视频平台上的免费电影

✅ **电影信息展示** - 显示电影标题、时长、简介等详细信息

## 注意事项

1. 目前仅支持腾讯视频（v.qq.com）的链接
2. 某些受版权保护的视频可能无法正常嵌入播放
3. 视频播放需要网络连接
4. 免费电影推荐功能依赖于网络爬虫，可能会受到网站反爬机制的影响
5. 首次使用免费电影功能时，系统需要一定时间来爬取数据
